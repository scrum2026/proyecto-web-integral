# Proyecto web integral

Estructura inicial para un proyecto web (HTML, CSS y JavaScript).

## Estructura

| Carpeta / archivo | Descripción |
|-------------------|-------------|
| `public/` | Archivos estáticos públicos (favicon, imágenes servidas tal cual). |
| `src/css/` | Hojas de estilo. |
| `src/js/` | Scripts del cliente. |
| `src/assets/` | Recursos internos (iconos, fuentes, etc.). |
| `docs/` | Documentación del proyecto. |
| `index.html` | Punto de entrada de la aplicación. |

## Cómo abrir el proyecto

Abre `index.html` en el navegador o sirve la carpeta raíz con un servidor local (por ejemplo: `npx serve .`).
